# Pratik Sonigra — Portfolio

Personal portfolio website built with Next.js 15 and Tailwind CSS v4.

## Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Language**: TypeScript
- **Deployment**: Vercel

## Features

- Dark glassmorphism design with cyan accent
- Fluid mouse-tracking glow effects
- Bento-grid project showcase
- Fully responsive (mobile, tablet, desktop)
- Smooth scroll navigation

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it locally.
